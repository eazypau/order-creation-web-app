# Order Creation Web App

A website to create and manage your orders.

## Technologies Used

-   NextJs
-   React
-   TypeScript
-   headlessui
-   trpc
-   clsx
-   zod
-   eslint
-   husky
-   linted-staged
-   tailwindcss
-   prettier
-   postgres
-   prisma

## Features

-   create order
-   update order
-   create product
-   update product
-   delete product
-   language translation

# Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and preview
$ npm run build
$ npm run start
```
